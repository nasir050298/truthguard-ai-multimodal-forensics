# TruthGuard AI — Multimodal Deepfake Detection & Forensics Platform

TruthGuard AI is a unified React + TypeScript AI forensics platform that combines the best ideas from three separate projects: DeepFakeShield, VideoTruth, and VoiceGuard AI. The final project turns image, video, and voice detection into one professional multimodal AI authenticity platform.

## Demo Accounts

- Analyst: `analyst@truthguard.ai` / `demo123`
- Reviewer: `reviewer@truthguard.ai` / `demo123`
- Admin: `admin@truthguard.ai` / `demo123`

## Features

### Public Website
- Modern AI security landing page
- Features, use cases, models, pricing, docs, and contact pages
- Responsive dark cyber-forensics design

### AI Forensics Dashboard
- Unified command center
- Image, video, and voice analyzers
- Multimodal trend charts
- Risk distribution charts
- Recent case results

### Detection Modules
- Image forensics with artifact/metadata/attention-overlay concepts
- Video forensics with frame-risk timeline and temporal manipulation concept
- Voice forensics with waveform, artifact, voiceprint, and replay-risk concepts

### Workflow
- Reports table
- Case review queue
- Model lab and custom model slot
- Threat intelligence
- Audit logs
- Responsible AI limitations
- Platform settings
